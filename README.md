# LAB №5 REPORT

## Задание:

Запустить программу из ЛР№3 на суперкомпьютере.

## Исходный код

Представлен в файлах main.cpp, Matrix.h для языка C++ и в файле верификации calculation_verification.py для языка Python  

Для удобного запуска кода на суперкомпьютере написан специальный  
скрипт [SpartMPI.pbs](StartMPI.pbs) , который содержит в себе информацию о необходимом модуле intel/mpi4 и настройки к запуску кода

## Результаты экспериментов:

Суперкомпьютер выполнил перемножение квадратных матриц 1000х1000 за $1.13923$ секунд, что дольше $1.08$ секунд на моём устройстве. [Выход суперкомпьютера](sk_out/slurm-252865.out)  
Ради интереса проведен эксперимент для матриц 4000 порядка на 16 процессорах. Результаты представлены в [выходном файле](sk_out/slurm-252874.out). Время вычисления составило $37.2116$ секунд.
